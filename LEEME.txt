flask run

# Debugging en Flask

Debugging: es el proceso de identificar y corregir errores de programación.

Para activar el debug mode escribir lo siguiente en la consola:

export FLASK_DEBUG=1
echo $FLASK_DEBUG

En windows:

$env:FLASK_DEBUG=1

opción para correr el servidor en modo debug es poner estas lineas al final del archivo main.py:

if __name__ == '__main__':
    app.run(debug=True)

o

if __name__ == '__main__':
    app.run(port = 5000, debug = True)


y correr main.py desde la terminal:

python main.py
